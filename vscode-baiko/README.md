# Baiko — VS Code Extension

Fitohana ho an'ny fiteny fanaovana programa **Baiko** (fiteny malgasy).

## Endri-kasa

- Fanamarihana syntaxe (syntax highlighting)
- Fanampin-teny (autocompletion) — teny fototra, variables, fonksiona
- Hover documentation
- Signature help
- Go to definition
- Diagnostic errors (soratra mena eo amin'ny hadisoana)

## Fampiasana

Sokafy ny rakitra `.baiko` ka hiasa avy hatrany ny fitohana.
